Web Designer Keisuke Kagoshima

千葉県在住のWebデザイナー、鹿子嶋啓介です。オンラインスクールでWebディレクションとフロントエンドについて学び、DTP広告業界から憧れていたWebデザイナーへと転職活動中です。これからさらにWeb制作の技術や知識を身につけて、Webサイト運用や問題解決のアドバイスもできるデザイナーになりたいと思っています。

Skills

Web

Web UIデザインはフリーとして業務を請け負っていますが、Dreamweaverでのコーディングもできます。HTML/CSS,JavaScript、PHP、WordPressは現在勉強中です。

Illust

Illustrator,PhotoshopやCinema 4Dを使用したキャラクターデザインやパースなどのイラスト制作が得意です。本の挿絵なども描いています。

English

22歳の頃にオーストラリアで滞在、アルバイトをしながら暮らしていた時期がありましたので英語が得意です。今でも仕事の情報収集に役立てています。

Direction

広告代理店制作部のチーフグラフィックデザイナーとして、長年ディレクション業をこなしていた経験があります。全体のバランスを見ながらの進行業務や細かい問題解決には慣れています。
